# 💊 AI-Based Drug–Drug Interaction (DDI) Prediction System

## 📌 Overview
The AI-Based Drug–Drug Interaction (DDI) Prediction System is a Machine Learning project that predicts potential interactions between two drug molecules using their chemical structure in SMILES format.

This system helps in identifying harmful drug combinations at an early stage, supporting safer drug development and better clinical decision-making.

---

## 🚀 Features
- 🔬 Predicts interaction between two drugs
- 🧪 Accepts SMILES format as input
- 🧠 Machine Learning-based prediction
- ⚗️ Molecular fingerprint generation using RDKit
- 💻 Interactive web application using Streamlit
- ⚡ Fast and user-friendly interface

---

## 🧪 Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- RDKit
- Streamlit
